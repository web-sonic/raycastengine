# RayCast Engine
Simple 3D game engine

1. <code>make</code>
2. <code>./cub3d subject_map.cub</code> or <code>./cub3d maps/subject_map.cub --save</code> to create screenshot

### Control

<p>w - forward</br>
<p>s - back</br>
<p>a - left</br>
<p>d - right</br>
<p>&#8592 - rotate left</br>
<p>&#8594 - rotate right</p>
