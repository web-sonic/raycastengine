# RayCast Engine
Simple 3D engine

1. <code>make</code>
2. <code>./cub3d subject_map.cub</code> or <code>./cub3d subject_map.cub --save</code> to create screenshot

### Control

<p>w - forward</p>
<p>s - back</p>
<p>a - left</p>
<p>d - right</p>
<p>&#8592 - rotate left</p>
<p>&#8594 - rotate right</p>
