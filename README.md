### RayCast Engine
Simple 3D engine

1. <code>make</code>
2. <code>./cub3d subject_map.cub</code> or <code>./cub3d subject_map.cub --save</code> to create screenshot

# Control

w - forward
s - back
a - left
d - right
&#8592 - rotate left
&#8594 - rotate right
